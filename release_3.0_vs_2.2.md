# SlipSnap 3.0 vs 2.2

## Русский

Новые функции:

- Быстрый шаринг из редактора по ссылке (24ч).
- Улучшенный GIF-процесс: вставка анимированных GIF и корректный экспорт анимации.
- Zoom Lens (лупа) как отдельный инструмент.
- Анимации объектов: `Draw` и `Pulse`.
- Обновленная библиотека мемов: анимированные превью, удаление, переименование, открытие папки.
- Улучшенный видеозахват и экспорт в MP4/GIF.
- Более стабильный захват на нескольких мониторах.

## English

New features:

- Quick editor sharing via temporary link (24h).
- Improved GIF workflow: paste animated GIFs and export animations correctly.
- New Zoom Lens tool.
- Object animations: `Draw` and `Pulse`.
- Updated meme library: animated previews, delete/rename, open folder.
- Improved video capture and MP4/GIF export.
- More reliable capture on multi-monitor setups.
